What to Expect When Downloading

Because HeartStor ChMS is independent and not digitally signed by a large corporation, your computer may show one or more of the warnings below. These are common for new or community software.

1. Browser Warning (Chrome / Edge)

After the download completes, your browser might display a message such as "This file is not commonly downloaded and may be dangerous."

The fix:

1. Look for the downloaded file at the bottom of the browser window or in the Downloads panel.

2. Click the arrow or three dots next to the file.

3. Select Keep. If prompted again, choose Keep anyway.

2. Antivirus Flags

Some aggressive antivirus tools flag all newly downloaded .exe files as "Low Reputation".

The fix:

1. If your antivirus blocks the installer, open your antivirus notifications.

2. Look for the blocked HeartStor installer entry.

3. Choose the option to Allow, Restore, or Trust the file.

4. If needed, temporarily add the installer to your exclusions list while you install.

How to Bypass Windows SmartScreen

Windows SmartScreen helps block unknown apps. Because HeartStor ChMS is from an independent developer, it may show an extra confirmation screen the first time you run it.

Step-by-Step: Running the Installer

1. Locate the downloaded file (for example, HS64_Setup.exe or HS32_Setup.exe) in your Downloads folder.

2. Double-click the file to run it.

3. If you see a blue window titled "Windows protected your PC", look for a small link that says "More info".

4. Click "More info". This will reveal a button labeled "Run anyway".

5. Click "Run anyway" to start the HeartStor installer.

Why You Are Seeing These Warnings

Microsoft and Google prioritize software from companies that purchase Code Signing Certificates, which require ongoing annual fees.

As an independent developer providing HeartStor ChMS for free, it is not practical to pay for these certificates. Your computer therefore labels the app as coming from an "Unknown Publisher", even though the file itself is safe.

These warnings are about reputation, not about any known harm in the file.
